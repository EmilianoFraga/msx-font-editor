1- Compilando e executando o MSX Font Editor

O MSX Font Editor foi criado no ambiente Java, no qual é compatível com diversos sistemas operacionais como Windows, Linux e Mac.
Não é necessário recompilar o código executável do programa para rodar em outro sistema operacional, porque aplicativos Java são multi-plataforma.

1.1. Compilando
* No diretório dos códigos-fonte do programa (*.java), digite na linha de comando:
javac Main.java

1.2. Criando pacotes:
* Copie todos os arquivos ".class" gerados na compilação para um diretório vazio.
* Copie os arquivos de imagem (img) e o "META-INF" (presente no arquivo dos executáveis) para esse mesmo diretório.
* Compacte todo os arquivos e sub-diretórios desse diretório com o ZIP.
* Renomeie a extensão do arquivo comprimido de ".zip" para ".jar".

1.3. Rodando o arquvo ".jar"
* Digite na linha de comando:
java -jar mfe.jar
